Информация по тегам и примеры их использования

Теги <dl>,<dt>,<dd> и описание к ним


Данные теги предназначены для создания списка определений.
Данные списки подходят для форматов:вопрос-ответ;наименование и определение;категория и тема.
Списки состоят из трех тегов.

  • <dl>(сокращение от «description list») обозначает сам список описаний;
  • <dt>(сокращение от «description term») обозначает термин;
  • <dd>(сокращение от «description definition») обозначает описание или определение.

Теги <dt> и <dd> пишутся внутри <dl>

Пример использования

растровый формат хранения графической информации, использующий сжатие без потерь по алгоритму Deflate.
UNICODE
Cтандарт кодирования символов,включающий в себя знаки почти всех письменных языков мира.
В настоящее время стандарт является преобладающим в Интернете.

Данный тег определяет блок предварительно форматированного текста.

Тег <code>

Тег <code> предназначен для отображения одной или нескольких строк текста, который представляет собой программный код.

Пример использования


Код программы

while (src != null) {
if (src.tagName == dest) return src
src = src.parentElement
}
return null
}

Тег <sub> и <sup>

Тег <sub> отображает шрифт в виде нижнего индекса.
Текст при этом располагается ниже базовой линии остальных символов строки и уменьшенного размера.
Тег <sup> отображает шрифт в виде вернего индекса.
Текст при этом располагается выше базовой линии остальных символов строки и уменьшенного размера.

Пример использования

Оксид натрия Na2O

Тег <q>

С помощью этого тега мы можем выделить какую-то цитату в тексте.В браузере мы увидим обычные кавычки

Тег <blockquote>

Данный тег позволяет выделить длинные цитаты внутри документа.
Текст,обозначенный этим тегом,отображается как выровненный блок с отступами слева и справа,а также с отбивкой сверху и снизу.
С данным тегом также может использоваться тег cite,чтобы пометить текст как сноску на другой материал

Тег <nobr>

Тег nobr уведомляет браузер отображать текст без переносов. Использование тега nobr заставляет отображать текст без переносов,одной строкой,что может привести к появлению горизонтальной полосы прокрутки.
Пользователям придется прокручивать текст по горизонтали,чтобы увидеть его полностью

Тег <wbr>

Тег <wbr> (англ. word break opportunity) предоставляет возможность указать браузеру,что при необходимости в этом месте можно сделать перенос строки(например,слишком длинное слово или адрес URL).
Отличие тега wbr от br заключается в том,что в месте расположения тега br браузер обязательно переносит содержимое на новую строку,а встречая в коде элемент wbr только по необходимости.
